Class Overview

This receiver will receive all requests events, errors and results if host android.app.Activity can be recreated during the process. You should create a child of this class in your android.app.Activity or android.app.Fragment, and register its instance on onStart() and onStop() methods. Then you can run requests using VK.ReceiverCallback.
